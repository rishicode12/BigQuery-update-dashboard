# BigQuery-update-dashboard

A modern dashboard for tracking the latest BigQuery release updates directly from official Google Cloud release feeds. The application automatically fetches, parses, categorizes, and displays BigQuery release notes in an easy-to-navigate interface while providing tools for sharing updates on X (Twitter).

## 🚀 Overview

BigQuery-update-dashboard helps developers, data engineers, and cloud practitioners stay informed about the latest BigQuery features, fixes, and announcements without manually browsing release notes.

The dashboard fetches release information from official Google Cloud XML/RSS feeds, organizes updates into categories, and provides search, filtering, analytics, and social sharing capabilities.

---

## ✨ Features

### 📡 Live XML Feed Fetching & Parsing

* Fetches official BigQuery release notes from Google Cloud RSS/XML feeds.
* Parses XML content automatically.
* Splits release entries into individual update cards using heading detection.
* Displays updates in a clean and readable format.

### 🏷️ Automatic Category Classification

Updates are automatically categorized into:

* 🚀 Features
* ⚠️ Issues & Fixes
* 📢 Announcements
* ⚡ Others

This enables faster discovery of relevant changes.

### 🔍 Interactive Feed Controls

* Real-time keyword search.
* Category-based filtering.
* Sidebar navigation.
* Statistics dashboard showing update counts by category.

### 🐦 Multi-Tweet Composer

Create social media summaries directly from release notes.

Features include:

* Multi-card selection.
* Automatic tweet generation.
* Multiple formatting styles:

  * Digest
  * Bulleted
  * Headline
* One-click Copy functionality.
* Direct "Tweet on X" integration.

### 🎨 Theme Management

* Light Mode
* Dark Mode
* Persistent theme preferences using client-side storage.
